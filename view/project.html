初期化

  1. Gitリポジトリを作成
  2. ※ リポジトリ名がプロジェクト名となるので注意
    • GitHub の以下のボタンから
  3. ローカルにリポジトリをクローン
    • このディレクトリがプロジェクトのルートディレクトリとなる。
  4. ルートディレクトリで「npm init -y」を実行
    • 実行後、package.json が自動生成される。

gulp導入

  1. ルートディレクトリで以下のコマンドを実行
    • npm --save-dev install gulp
    • npm --save-dev install gulp-babel@5.3.0
    • npm --save-dev install gulp-load-plugins
    • npm --save-dev install gulp-util
    • npm --save-dev install through2
    • npm --save-dev install browserify
      • gulpを用いた基本的なプログラミングで利用
      • babelのバージョン指定はelectronとの互換性のため
    • npm --save-dev install gulp-mocha
    • npm --save-dev install mocha
      • ユニットテストで利用
    • npm --save-dev install electron-prebuilt
    • npm --save-dev install electron-packager
    • npm --save-dev install asar
      • electronアプリケーションの開発で利用
  2. ルートディレクトリ直下に生成される node_modules ディレクトリを、.gitignoreに登録
    • ディレクトリを右クリックして「TortoiseGit」→「無視リストに追加」

    • index.js (e-freak向け)
  2. ※ 内容は適宜調整のこと

ディレクトリ構成

  1. 以下の構成を推奨しておく
    • [root]
      • node_modules
      • app
        • css
          • default.css
        • image
        • script
        • sound
        • view
          • xxx.html
          • yyy.html
          • zzz.html
      • cache
      • src
        • main
          • jsx
            • aaa.jsx
            • bbb.jsx
            • ccc.jsx
        • test
          • jsx
            • aaa-test.jsx
            • bbb-test.jsx
            • ccc-test.jsx
      • .gitignore
      • gulpfile.js
      • index.js
      • package.json
  2. ※ cache ディレクトリについては、.gitignoreに登録しておく
    • ディレクトリを右クリックして「TortoiseGit」→「無視リストに追加」

Hello, World

  1. app/view ディレクトリ直下に title.html を作成する
  2. title.html をテキストエディタで開き、「Hello, World」とだけ書いて保存
  3. ルートディレクトリで「electron [プロジェクト名]」を実行
    • よく使うことになるので、上記コマンドを記載したbatファイルでも作っておくと良い
  4. electronのウィンドウが表示されたら成功
